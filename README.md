## Tools used

HTML, CSS, JavaScript, Bootstrap, jQuery, Flask

## Steps to run the program
Obtain and set your API key
```
$ vim openai_secrets.py
```
Download the repository and run server.py
```
$ python3 server.py
```
Go to http://127.0.0.1:5000 in your browser
